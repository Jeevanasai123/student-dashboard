# student-dashboard
Task 3
A responsive web dashboard to manage student data, attendance, and subjects.
Features a clean UI with statistics, student lists, and attendance tracking. Built with HTML, CSS, JavaScript, and optionally integrates with backend services for dynamic data.

#Features

Dashboard with summary statistics (students, attendance)

Student list table with status indicators

Attendance tracking (present/absent)

Sidebar navigation (Dashboard, Students, Teachers, Subjects, Settings)

Responsive design for desktop and mobile
